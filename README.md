# Receipts for Lovely Loveseats

A Python program designed to catalog furniture descriptions, track item prices, handle sales tax calculation, and print itemized receipts for customers. This project was completed as part of the **Codecademy Data Engineer Career Path** to master core Python fundamentals such as variable assignment, string manipulation, and arithmetic operations.

---

## 🚀 Project Overview

The script simulates a point-of-sale transaction for a home furnishings boutique named *Lovely Loveseats*. It dynamically manages inventory metadata and automates the checkout process by calculating itemized descriptions, subtotal accumulation, local sales tax addition, and final billing totals.
